# Расширения

в данной директории представлена полная коллекция расширения Шиори.
Все расширения поделена на категориям:

```
- contrib:    Связь с внешними сервисами, использование сторонних API.
- eco:        Экономическая система и связанные с ней расширения.
- guild:      Расширения для вашего сервера.
- mini_games: Различные небольшие игры, как для одного, так и для компании.
- profile:    Дополнительный функционал пользователя.
- system:     Управление Шиори.
- templates:  Пример реализации расширений.
- utils:      Самые резные небольшие расширения.
- voice:      Расширения для голосовых каналов.
```
